Uma linguagem feita para conectar ecossistemas.

Klang não tenta substituir linguagens — ela existe para conectá-las com uma sintaxe clara, minimalista e totalmente determinística.

Filosofia

  • Clareza acima de tudo.
  • Semântica determinística.
  • Transpilação modular e previsível.
  • Interoperabilidade real com Java, C, Go, Rust e Python.

Exemplo de Sintaxe

@Use("java")
public static integer sum(integer a, integer b) {
    if (a > b) {
        return a;
    } otherwise (a == b) {
        return 0;
    } afterall {
        return b;
    }
}

Por que Klang?

Klang oferece um modelo simples: você escreve em uma sintaxe única, e transpila apenas o módulo desejado para o backend desejado. Isso mantém sistemas grandes **coerentes**, **claros** e **interoperáveis**.

K — Klang Language

Slogan: Uma Sintaxe, Qualquer Sistema.

Definição

K (de Klang Language) é uma linguagem criada para conectar linguagens e tornar a experiência de programar mais confortável e moderna. Seu objetivo não é gritar, mas ressoar — trazer harmonia, clareza e fluidez entre sistemas diferentes.

Essência

  • Conforto e harmonia na escrita de código
  • Interoperabilidade entre linguagens
  • Sintaxe clara e expressiva

Exemplo simples


@Use("java") // Indica a linguagem a ser utilizada.
public void main() { // Método principal a ser chamado, o core do programa.
    println("Olá Mundo.");
}

Dica: deslize horizontalmente no mobile para ver linhas longas.

K vem de Klang — soa como Clang, mas é mais suave. Uma linguagem feita para ressoar entre ideias, pessoas e sistemas.
